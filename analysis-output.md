# Import Dependency Analysis — src/ (excluding **tests**)

## Methodology

For each `.ts`/`.tsx` file under `src/` (excluding `__tests__/`), I searched for all static and dynamic imports referencing that file across the entire codebase (including test files). Both `from './...'` and `from '../...'` patterns were checked. Re-exports via `export { ... } from '...'` count as imports.
